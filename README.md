Project Setup

Clone the repo
git clone https://github.com/mohangurjar82/ProjectManagement-demo.git

cd path/to/project

Install dependencies: bundle install

Set up your database.yml file according to your credential

#After set up Run

rails db:create
rails db:migrate
rails db:seed

Start the Rails server: rails s -p 3001
